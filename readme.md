# ROUGE LITE 101

A Unity multiplayer game demo using AWS GameLift.

## Features
- Top-down movement and collectibles
- Multiplayer with Unity Netcode
- AWS GameLift server integration

## Getting Started

1. **Clone this repo:**  
   `git clone https://github.com/Bright-04/RougeLite101.git`
2. **Open in Unity 2022 LTS**
3. **Install dependencies** (see `/Docs/` or below) (To Be Added)
4. **Run the game** in the Unity Editor
