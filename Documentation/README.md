### For Level Designers

1. Start with [Manual_Tilemap_Room_Guide.md](./Level%20Design/Manual_Tilemap_Room_Guide.md) for a complete walkthrough
2. Use [Tilemap_Quick_Reference.md](./Quick%20References/Tilemap_Quick_Reference.md) for quick lookup during design
3. Utilize the Room Creation Helper tool in Unity (`Tools → Dungeon → Room Creation Helper`)

### For Programmers

3. Review the System Architecture to understand how everything connects

## 🛠️ Development Tools

The project includes several Unity Editor tools to assist with development:

- **Room Creation Helper** (`Tools → Dungeon → Room Creation Helper`)
  - Automated room structure setup
  - Quick spawn point configuration

- **Room Template Validator** (`Tools → Dungeon → Validate Room Templates`)
  - Validate existing room prefabs
  - Auto-fix missing components
